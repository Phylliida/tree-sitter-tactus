# tree-sitter-tactus

Tactus grammar for [tree-sitter](https://github.com/tree-sitter/tree-sitter).

Tactus is a verified Rust dialect that uses Lean's proof kernel for verification instead of Z3. It keeps Verus-style specs (requires/ensures/invariant) and replaces SMT assert chains with Lean-style tactic proofs.

Based on [tree-sitter-verus](https://github.com/verus-lang/verus).
